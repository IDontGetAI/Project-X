### 一、计算机系统的工作原理

#### 1.从C语言源程序到可执行文件 ﻿00:06﻿

- C语言源程序以.c后缀结尾，例如hello world程序
- 预处理器处理宏定义：将代码中的宏常量（如π= - 1415926）替换为实际值
- 编译器将C语言程序翻译为汇编语言程序
- 汇编器将汇编语言程序转换为机器语言目标模块（.o文件）
- 链接器整合目标模块和标准库函数，生成最终可执行文件（.exe）
- 可执行文件存储在外存（硬盘）中，需调入主存才能运行

#### 2.计算机系统的工作原理 ﻿02:23﻿

| 组件     | 功能                       | 交互关系             |
| -------- | -------------------------- | -------------------- |
| 硬盘     | 存储可执行文件             | 程序调入主存后运行   |
| 主存     | 临时存储运行程序           | 接收外存调入的程序   |
| CPU      | 执行程序指令               | 从主存读取指令和数据 |
| 输入设备 | 与程序交互（如键盘、鼠标） | 提供用户输入         |
| 输出设备 | 显示程序结果（如显示器）   | 输出运行结果         |

##### 1) 存程序工作方式 ﻿02:58﻿

- 程序由指令和数据组成，均以二进制形式无差别存储在主存中
- 每条指令/数据对应唯一主存地址
- 程序计数器（PC）寄存器指向下一条待执行指令，执行后自动+1
- 指令执行本质：从内存/寄存器获取数据→进行运算→写回结果
- 重点掌握概念：编译器、汇编器、链接器的作用及工作流程
