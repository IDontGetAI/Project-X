##### 1) 主存储器的基本部件 ﻿00:31﻿

- 存储体：由存储元件构成，用于存放二进制数据。 
- MAR（存储地址寄存器）：存放待访问数据的地址信息。 
- MDR（存储数据寄存器）：存放从存储体读出或待写入的数据。 
- 寄存器功能：MAR与MDR分别管理地址与数据，协同完成读写操作。 

##### 2) 主存储器与菜鸟驿站模型的类比 ﻿01:39﻿

| 主存储器部件 | 菜鸟驿站对应物 | 功能类比          |
| ------------ | -------------- | ----------------- |
| 存储体       | 货架           | 存放数据/包裹     |
| MAR          | 取件号         | 定位数据位置      |
| MDR          | 柜台           | 临时存放数据/包裹 |

- 差异点：主存储器支持读写双向操作，而菜鸟驿站仅支持取件。 

##### 3) 存储体的原理 ﻿05:31﻿

- 存储体结构：数据按地址存储，每个地址对应一个存储单元。 

##### 4) 存储单元、存储字、存储字长的概念 ﻿05:46﻿

- 存储单元：存储体的最小可寻址单位，存放一串二进制代码（存储字）。 
- 存储字长：每个存储单元容纳的二进制位数（如8/16/32/64位）。 
- 地址分配：存储单元地址从0开始，由MAR指定。 

##### 5) 存储元件与MAR、MDR的关系 ﻿07:04﻿

- 存储元件：基于电容原理，单个元件存储1比特数据。 
- MAR位数决定存储单元数量：若MAR为4位，则存储单元数为2^4=16个。 
- MDR位数等于存储字长：如MDR为16位，则每个存储单元存放16比特数据。 

##### 6) 字与字节、大B与小b的区别 ﻿09:01﻿

- 字（Word）：存储字长，具体位数由硬件设计决定（如16/32位）。 
- 字节（Byte）：固定为8比特。 
- 单位符号： 
  - 大B（Byte）：字节（如下载速度10MB/s）。 
  - 小b（bit）：比特（如宽带100Mb/s）。 

#### 3.运算器的基本组成 ﻿11:15﻿

- 核心部件： 
  - ACC（累加器）：存放操作数及运算结果。 
  - MQ（乘商寄存器）：专用于乘除法操作。 
  - X（通用寄存器）：存放临时操作数。 
  - ALU（算术逻辑单元）：实现算术与逻辑运算的复杂电路。 
- 功能分工：寄存器暂存数据，ALU执行计算。 

#### 4.控制器的基本组成 ﻿13:50﻿

- 核心部件： 
  - CU（控制单元）：分析指令并发出控制信号。 
  - IR（指令寄存器）：存放当前执行的指令。 
  - PC（程序计数器）：存放下一条指令地址，自动加1。 
- 指令执行阶段： 
  - 取指令：根据PC地址从内存读取指令至IR。 
  - 分析指令：CU解析指令内容。 
  - 执行指令：CU协调其他部件完成操作。

  - PC内容通过地址总线传至MAR（地址寄存器），主存根据MAR地址读取0号单元数据至MDR（数据寄存器）。 
  - MDR中指令经数据总线存入IR（指令寄存器）。 
- 分析指令： 
  - 操作码（前6比特）送CU（控制单元），解析为取数指令（操作码0001）。 
  - 地址码（后10比特）指明变量a的存储地址（5号单元）。 
- 执行指令： 
  - 主存读取5号单元数据（a=2）至MDR，再传至ACC（累加寄存器）。 
  - 关键效果：PC自动加1指向下一条指令（1号单元），ACC存入a的值（2）。

- 主机三大部件： 
  - 主存：存储体涉及字与字节的区别、大B（Byte）与小b（bit）的区别（高频考点）。
