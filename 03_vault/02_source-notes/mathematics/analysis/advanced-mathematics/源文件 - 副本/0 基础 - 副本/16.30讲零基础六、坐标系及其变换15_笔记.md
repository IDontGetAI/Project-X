- **坐标轴的旋转** 21:40

设旧坐标系旋转角度 $\theta$ 得到新坐标系，点 $M$ 在原坐标中为 $(x, y)$，新坐标中为 $(X, Y)$：

| 参数 | 表达式 |
| :--- | :--- |
| 原坐标 $x$ | $X\cos \theta - Y\sin \theta$ |
| 原坐标 $y$ | $X\sin \theta + Y\cos \theta$ |
| 新坐标 $X$ | $x\cos \theta + y\sin \theta$ |
| 新坐标 $Y$ | $-x\sin \theta + y\cos \theta$ |

  - 新坐标 $\to$ 旧坐标：$\begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix} \begin{pmatrix} X \\ Y \end{pmatrix}$
此变换为**正交变换**。

- **例题：旋转变换应用** 34:07
已知新坐标系由旧坐标系绕原点逆时针旋转 $\pi/4$ 得到，原方程为 $xy = 1$。

#### 2. 极坐标系 38:25

##### 1) 极坐标系的定义 38:51
- **基本构成**：
  - **极点 $O$**：坐标系中心参考点。
  - **极轴 $OX$**：基准射线。
- **定位参数**：
  - **极径 $\rho$**：点到极点的距离 ($\rho \ge 0$)。
  - **极角 $\theta$**：射线与极轴正向的夹角 ($\theta \in [0, 2\pi)$)。

| 参数 | 定义 | 几何意义 | 取值范围 |
| :--- | :--- | :--- | :--- |
| 极径 $\rho$ | 点到极点距离 | 决定点的同心圆位置 | $\rho \ge 0$ |
| 极角 $\theta$ | 与极轴夹角 | 确定点的辐射方向 | $[0, 2\pi)$ |

- **狭义与广义**：狭义极坐标满足 $\rho \ge 0$ 且 $\theta \in [0, 2\pi)$，保证点与坐标一一对应。

- **极坐标曲线网**：由同心圆（$\rho = \text{const}$）和射线（$\theta = \text{const}$）构成。

##### 2) 常见曲线的极坐标方程 48:33

- **直线方程推导**：
  - 已知直线过点 $A(a, \alpha)$，倾斜角为 $\beta$。
  - 在 $\triangle OAM$ 中应用正弦定理：$\frac{\rho}{\sin(\alpha-\beta)} = \frac{a}{\sin(\theta-\beta)}$。
  - 简化形式：$\rho = \frac{a\sin(\alpha-\beta)}{\sin(\theta-\beta)}$。

- **圆的方程推导**：
  - 圆心 $(a, \alpha)$，半径 $r_0$。
  - **余弦定理**：$\rho^2 + a^2 - 2a\rho\cos(\theta-\alpha) = r_0^2$。
  - **特殊情形**：
    - 圆心在极轴上，且过极点 ($a = r_0, \alpha = 0$)：$\rho = 2r_0\cos \theta$。
    - 圆心在垂直极轴方向，且过极点 ($a = r_0, \alpha = \pi/2$)：$\rho = 2r_0\sin \theta$。

##### 3) 直角坐标与极坐标的关系 59:05

**转换公式**：
- **直角 $\to$ 极坐标**：
  - $x = \rho \cos \theta$
  - $y = \rho \sin \theta$
- **极坐标 $\to$ 直角**：
  - $\rho^2 = x^2 + y^2 \implies \rho = \sqrt{x^2 + y^2}$
  - $\tan \theta = y/x$

**示例**：
- 直角坐标 $(-3, 5)$：
  - $\rho = \sqrt{(-3)^2 + 5^2} = \sqrt{34}$。
  - $\tan \theta = -5/3$。由于点在第二象限，$\theta = \pi + \arctan(-5/3)$。

##### 4) 方程转换示例

| 极坐标方程 | 转换过程 | 直角坐标方程 |
| :--- | :--- | :--- |
| $\rho \cos \theta = 1$ | $x = \rho \cos \theta$ | $x = 1$ |
| $\rho^2 \sin 2\theta = 2$ | $2\rho^2 \sin \theta \cos \theta = 2$ | $2xy = 2 \implies xy = 1$ |
| $\rho^2 \cos 2\theta = 1$ | $\rho^2(\cos^2 \theta - \sin^2 \theta) = 1$ | $x^2 - y^2 = 1$ |
| $\rho = \frac{1}{1 + 2\cos \theta}$ | $\rho + 2\rho\cos \theta = 1$ | $\sqrt{x^2 + y^2} + 2x = 1$ |
| $\rho = 1 - \cos \theta$ | $\rho^2 = \rho - \rho\cos \theta$ | $x^2 + y^2 = \sqrt{x^2 + y^2} - x$ |

**补充案例**：
- **直角 $\to$ 极坐标**：
  - $x^2 + y^2 - 4y = 0 \implies \rho^2 - 4\rho\sin \theta = 0 \implies \rho = 4\sin \theta$。
  - $x + \sqrt{3}y = 4 \implies \rho \cos \theta + \sqrt{3}\rho \sin \theta = 4 \implies \rho \sin(\theta + \pi/6) = 2$。
- **指数函数**：$y = e^x \implies \rho \sin \theta = e^{\rho \cos \theta}$。
