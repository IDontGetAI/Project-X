### 一、数列及其单调性 00:02

数列是考研数学中的重要研究对象，零基础阶段需重点掌握数列极限的衔接内容。

#### 1. 数列的概念 00:41
- **数列定义**：对于每个正整数 $n$，按特定法则对应一个实数 $x_n$，按下标从小到大排列的序列称为数列，记为 $\{x_n\}$。
- **数列本质**：数列可视为自变量为正整数 $n$ 的函数，即整标函数。
- **无穷数列**：考研数学中讨论的数列均为无穷数列，这是研究极限的前提条件。

#### 2. 等差数列 03:04
- **定义**：首项为 $a_1$，公差为 $d$ ($d \ne 0$) 的数列。
- **通项公式**：$x_n = a_1 + (n-1)d$。
- **前 $n$ 项和公式**：
  - $S_n = na_1 + \frac{n(n-1)d}{2}$。
  - $S_n = \frac{n(a_1 + a_n)}{2}$。

#### 3. 等比数列 05:02

##### 1) 等比数列前 $n$ 项和公式 05:09
- **定义**：首项为 $a_1$，公比为 $q$ 的数列。
- **通项公式**：$x_n = a_1 q^{n-1}$。
- **前 $n$ 项和公式**：
  - 若 $q = 1$：$S_n = na_1$。
  - 若 $q \ne 1$：$S_n = \frac{a_1(1-q^n)}{1-q}$。

##### 2) 等比数列求和公式的推导 07:01
- **推导方法：错位相减法**。将 $S_n$ 与 $qS_n$ 相减消去中间项，最终得到求和公式。
- **关键步骤**：
  - $S_n = a_1 + a_1q + \dots + a_1q^{n-1}$。
  - $qS_n = a_1q + a_1q^2 + \dots + a_1q^n$。
  - 两式相减：$(1-q)S_n = a_1(1-q^n)$。

#### 4. 单调数列 09:37
- **定义**：
  - **单调不减**：$\forall n \in \mathbb{N}^+, a_{n+1} \ge a_n$。
  - **单调递增**：$\forall n \in \mathbb{N}^+, a_{n+1} > a_n$（严格单调）。
  - **单调不增 / 递减**：类比定义。
- **重要性**：单调性是判断数列极限存在性的关键性质。

##### 2) 例题：数列概念应用 18:10

##### 3) 例题：等比数列通项公式应用 20:42
数列 $\{a_n\}$ 中 $a_1 = 2$，$a_{n+1} = a_n + \ln(1 + 1/n)$，求 $a_n$ 的通项：
- **累加法**：$a_n = a_1 + \sum_{k=1}^{n-1} (a_{k+1} - a_k)$。
- **代入**：$a_n = 2 + \sum_{k=1}^{n-1} \ln(\frac{k+1}{k}) = 2 + [\ln 2 - \ln 1 + \ln 3 - \ln 2 + \dots + \ln n - \ln(n-1)] = 2 + \ln n$。
- **最终解**：$a_n = \ln n + 2$（答案选 A）。

##### 5) 例题：累乘法求解 28:27
已知数列 $\{a_n\}$ 满足 $S_n = \frac{n+2}{3} a_n$，求 $a_n$ 的通项：
- **建立递推**：$a_{n+1} = S_{n+1} - S_n = \frac{n+3}{3}a_{n+1} - \frac{n+2}{3}a_n \implies \frac{3-n-3}{3}a_{n+1} = -\frac{n+2}{3}a_n \implies -na_{n+1} = -(n+2)a_n \implies \frac{a_{n+1}}{a_n} = \frac{n+2}{n}$。
- **累乘法**：$a_n = a_1 \cdot \prod_{k=1}^{n-1} \frac{k+2}{k} = a_1 \cdot \frac{n(n+1)}{1 \cdot 2} = a_1 \cdot \frac{n(n+1)}{2}$。
- **最终通项**：$a_n = \frac{n(n+1)}{2}$（已知 $a_1=1$）。
