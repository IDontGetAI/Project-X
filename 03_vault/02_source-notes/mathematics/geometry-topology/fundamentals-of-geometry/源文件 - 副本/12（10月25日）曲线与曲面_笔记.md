- 参数方程：在空间中表示为﻿$\begin{cases} x = a_1 + t u_1 + s v_1 \ y = a_2 + t u_2 + s v_2 \ z = a_3 + t u_3 + s v_3 \end{cases}$﻿或向量形式﻿$\overrightarrow{OP} = \overrightarrow{OA} + t \overrightarrow{u} + s \overrightarrow{v}$﻿，其中﻿$(a_1,a_2,a_3)$﻿为平面经过的点，﻿$\overrightarrow{u}$﻿和﻿$\overrightarrow{v}$﻿为方向向量。

#### 2.直线方程：参数方程、一般方程、约化方程、标准方程 ﻿17:18﻿

直线方程的四种形式如下：

- 参数方程：﻿$\begin{cases} x = a_1 + t u_1 \ y = a_2 + t u_2 \ z = a_3 + t u_3 \end{cases}$﻿或向量形式﻿$\overrightarrow{OP} = \overrightarrow{OA} + t \overrightarrow{u}$﻿，表示过点﻿$(a_1,a_2,a_3)$﻿且方向为﻿$\overrightarrow{u}$﻿的直线。

重点：直线方程中参数方程和标准方程最常用，因其能直接读出几何信息（点和方向向量）。
