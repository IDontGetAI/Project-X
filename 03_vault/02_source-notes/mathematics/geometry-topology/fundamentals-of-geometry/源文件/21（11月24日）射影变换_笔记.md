##### 2) 摄影变换的定义：基于可逆矩阵的线性映射 ﻿05:04﻿

三×三可逆矩阵的一个优势在于，从任意矩阵出发可定义一个线性映射。用LA表示该线性映射，其作用是从三维欧式空间映射到三维欧式空间。此映射可将任意向量x映射到ax。任意矩阵均可定义此类线性映射，但三×三可逆矩阵的特殊之处在于其映射具有可逆性。可逆映射不会出现不良情况，例如较差的线性映射会将整个三维空间压缩至二维空间，如投影映射将三维欧式空间投影至二维平面。若地面坐标系建立在地面上，对应的矩阵最下面一行和右边一列可能全为零，此时映射效果不佳，会将三维空间压缩。而三×三可逆矩阵的映射不会出现此类情况，其优势在于保持线性性质，即若将x乘以常数，所得结果也乘以常数，相当于将直线映射为直线。

#### 3.摄影空间的直线 ﻿13:04﻿

##### 1) 直线的定义

直线在摄影空间里的定义为：l等于x比，y比，z，满足ax + by + cz = 0。这是摄影框架中直线的定义。直线是用一次齐次多项式表示，即ax + by + cz = 0。该多项式关于x、y、z均为一次，齐次意味着所有项次数相同，既无二次项也无零次项，所有项均为一次，且没有常数项（常数项为零次）。齐次性在摄影空间中至关重要，若缺乏齐次性，对象无法在摄影空间中定义。通过上述方式定义直线。

##### 2) 直线的几何解释 ﻿14:16﻿

在几何中，已知在三维空间模型中，考虑z=1这个平面，若在该平面上有一条直线，这条直线在z=1平面上，对应的是ax+by+cz=0这个平面（该平面过原点）。过原点的平面与z=1平面相交，交线即为所讨论的直线。在空间中，一个黄色平面与白色平面（z=1平面）相交，交出的线为黄色直线。直观上，每一条过原点的直线可视为一个点，一组过原点的、同一方向的直线组成的集合，实际上构成一个平面。因此，射影空间中的直线，对应于原空间中的某一个平面。具体而言，射影空间中的元素都是过原点的线，在黄色平面中过原点的直线所组成的集合，即为射影直线（在射影几何中，通常直接称之为直线）。射影直线指一个平面里面过原点的直线的集合。

射影变换将平面映射为平面，这是欧式理论的一部分。具体表现为将直线映射为直线，将平面映射为平面，即将欧式空间中过原点的直线映射为过原点的直线，将欧式空间中过原点的平面映射为过原点的平面。结合上述解释，欧式空间中过原点的平面对应于射影直线。因此，射影变换不仅将射影空间的点映射为点，还将射影空间的直线映射为直线（可能将一条直线映射为另一条直线，因为可能将一个平面映射为另一个平面）。这是射影变换的一个基本性质，该性质源于可逆性。

射影空间中的几何与正常几何稍有不同。相同之处在于两点决定一条直线；不同之处在于两条直线决定一个点（交于一个点）。这两个命题是对偶的，即任何两点存在唯一的一条直线过这两个点，反过来，任何两条直线也存在唯一的一个点恰好落在这两条直线的交点上。这是点线的对偶关系。上节课通过线的定义证明这两个命题不仅对偶，而且证明之间也完全对偶。

点线对偶关系的源头在于：给定一个点p（a:b:c），可对应于一条直线l（ax+by+cz=0），称为射影直线，记为lp；反过来，给定一条直线l（ax+by+cz=0），也对应一个点p，记为PL。点p与直线l之间存在一一对应关系。在对应过程中，一般情况下，点p在直线l上，当且仅当直线l所对应的点PL在点p所对应的直线lp上。利用这个定义可以简单证明上述性质，且利用这个性质可以推出对偶命题。即证明一个对偶命题后，利用该性质可推出另一个命题，反之亦然。但并非所有命题都可用此方式转化。

这两个性质是球面几何中测地线性质的基本推论。下面回到射影变换的角度继续讨论。

- 齐次化与摄影化流程 ﻿01:15:46﻿
  - 歧视化，或称摄影化。对于平面里的几何对象，平面内由代数方程所给定的内容，进行齐次化。齐次化指将原本非齐次的方程，即含一次项和零次项的方程，通过特定方式转化为全部为齐次项的方程。摄影化指将原本在欧式空间平面内的对象，转化为摄影平面内的对象，即将非摄影对象变为摄影对象。
  - 标准过程：摄影空间内的所有对象实质上均为比例关系。令﻿$x = \frac{x}{z}$﻿，﻿$y = \frac{y}{z}$﻿，选取这两个变量代入原方程，得到﻿$\frac{ax}{z} + \frac{by}{z} + c = 0$﻿。整理后得到﻿$ax + by + cz = 0$﻿，此即为齐次方程的来源。若仅通过乘以常数将方程变为齐次形式，所得结果将与原对象不对应。
  - 流程：从欧式空间内的代数方程出发，只要该方程为多项式方程，均可通过上述方式将其转化为齐次方程，从而得到摄影空间内的对象。
  - 方程的好处：﻿$l_0$﻿与﻿$l$﻿的差别在于，从空间视角观察，﻿$l_0$﻿为一条直线。在摄影空间中，若将摄影平面视为三维空间直线的集合，或从﻿$z=1$﻿平面视角观察并添加无穷远点（无穷远点分别对应平面上的各个方向），则该直线对应某一方向，该方向对应的无穷远点被添加到直线上后，得到摄影直线。摄影化过程即将直线对应的无穷远点加入，得到摄影直线。
  - 高次方程：一般的高次方程，特别是二次方程，均可进行类似操作。

  - 相交直线的分析本质上可得到两条直线。每一条直线通过添加其所在平面的无穷远方向，可得到一个射影直线，此部分与前述内容一致。接下来需明确的是椭圆、双曲线和抛物线的相关情况。未添加任何元素时，本质上一个点仍为一个点。添加一个无穷远点的情况对应一种曲线形态；添加两个无穷远点的情况对应另一种曲线形态。两条平行线对应的无穷远为同一个，因此仅添加一个无穷远点。
- 椭圆 ﻿01:30:51﻿
  - 椭圆在反射变换下本质上可视为x² + y² = 1的圆。在特定仿真条件下，此类图形经射影化后得到x² + y² - z² = 0的方程。该方程对应的曲线需分析其无穷远点特性：圆或椭圆本身不包含无穷远点。从几何角度验证，无穷远点对应z = 0的点。将z = 0代入方程，可得x = y = 0，此时x、y、z均等于零，不构成有效点，因此该曲线不存在无穷远点。此过程无需额外添加无穷远点。
- 抛物线 ﻿01:32:25﻿
  - 抛物线的表示形式，例如y=x²，可改写为x²-y=0。若对x²-y=0进行射影化处理，将得到x²-yz=0。从这一过程中分析是否存在无穷远点，无穷远点定义为z=0的点。当z=0时，x=0且y≠0，因此存在一个无穷远点，其坐标为(0,1,0)，表明该抛物线在射影化后增加了一个无穷远点。
- 双曲线 ﻿01:33:18﻿
  - 双曲线的方程为x²-y²=0，摄影化后变为x²-y²+z²=0。当常数项为零，且前面系数为零时，得到此方程。令z=0，则x要么等于y，要么等于负y，即x=y或x=-y。因此，存在两个无穷远点，即（1,1,0）和（1,-1,0）。
  - 从几何角度理解，椭圆无法达到无穷远点。对于抛物线，其方向逐渐接近某一特定方向，如y=x²的抛物线，其方向接近于y轴方向（0,1），即x=0，y方向。当考虑无穷远直线时，抛物线在无穷远直线上交于一点，因为对应的是同一方向。
  - 对于双曲线，在平面上，其形状逐渐接近两条渐近线。当加入无穷远点后，会加入对应于这两条线的元素。若将无穷远表示在一条直线上，双曲线看起来有两只，且这两只在无穷远处交于两点。
  - 这三个图形（椭圆、抛物线、双曲线）在加入无穷远点后，从某种角度看，形状变得相似。这一结论可以通过严格的证明得出，证明过程基于对方程的分析。

##### 3) 二次曲线的严格证明 ﻿01:36:35﻿

- 二次曲线方程与矩阵表示 ﻿01:36:36﻿
  - 二次曲线的方程由变量x、y、z及一个矩阵c构成，该方程用于表示二次曲线，此处矩阵c的具体形式不再赘述，因其已在前期内容中多次阐述。
- 摄影变换与可逆矩阵 ﻿01:36:51﻿
  - 方程在摄影变换下的操作：在摄影变换中，需乘以一个可逆矩阵。设可逆矩阵为a，作用于向量xyz时，需使用a的转置（因向量需转置为行向量形式）。 
  - 变换等价性：通过该变换，可将当前表达式与另一表达式建立等价关系。 
  - 核心步骤：后续操作与分类问题一致，即寻找满足atca条件的矩阵a。
- 正交阵与对角阵的转换 ﻿01:37:37﻿
  - 找正交阵可将其变为对角阵，变为对角阵后可进一步简化。本质过程是说明几种情况最终相同，即三种情况对应atca一定是兰普塔 - 兰普塔二，通过找正交阵可实现。这三种情况对应兰普特 - 兰普特 - 兰普特三非零的情况，前面所述对应某些等于零的情况，此处对应非零情况。
- 非零情况的进一步讨论 ﻿01:38:17﻿
  - 对于非零情况，已进行至特定程度，可继续操作至以下程度：例如λ₁（兰普塔一）非零时，可进行如下操作：若λ₁大于零，将等式左边乘以√λ₁，右边乘以1/√λ₁，可将该元素转化为1。对于大于零的情况，可将此类元素转化为1。需注意，λ₁、λ₂、λ₃（兰普塔 - 兰普塔 - 兰普塔三）不同号，分类讨论时，同号情况出现在特定条件下，即λ₁、λ₂、λ₃均为正时，重复操作后结果保持同号；不同号时，至少有一个为正，其余为负。若使用√(-λ₁)操作，会引入负号，因此可假定两正一负，最终可转化为[1, 1, -1]矩阵形式。若λ₁、λ₂、λ₃均非零且不同号，必存在矩阵将其转化为[1, 1, -1]形式。[1, 1, -1]矩阵对应欧式空间中的二次曲线起点。
- 圆锥曲线与二次曲面的关系 ﻿01:40:04﻿
  - 在欧式空间中，方程﻿$x^{2}+y^{2}-z^{2}=0$﻿表示一个圆锥。该圆锥以原点为顶点。
- 三类二次曲线的统一性 ﻿01:40:35﻿
  - 三类曲线全部在此处呈现，且均为等价关系，对应于圆锥曲线。 圆锥在三维空间中为二次曲面，在二维摄影平面中表现为二次曲线。 能够观测到的抛物线、椭圆等均为二次曲线的具体形式。
- 不同视角下的二次曲线 ﻿01:41:02﻿
  - 抛物线、双曲线、椭圆是通过不同角度的平面截取圆锥得到的。当用一个平面截取圆锥时，若平面与圆锥的轴线成一定角度且不平行于母线，会截出一个椭圆。当平面与圆锥的轴线夹角较小，且与母线平行时，会截出一个双曲线。当平面与圆锥的母线平行时，会截出一个抛物线。这三类曲线都是圆锥在不同角度下被平面截取后的射影结果，且截取时使用的平面均不经过圆锥的顶点。
  - 用不经过圆锥顶点的平面截取圆锥，得到的这三类曲线即为椭圆、双曲线、抛物线。
  - 关于圆锥与所截曲线的关系：本质上，所绘制的曲线是从不同平面视角观察圆锥截取结果的表现。例如，从上方平面观察，得到一种曲线；从侧面平面（如y=1平面）观察，得到另一种曲线。将平面内的圆置于射影空间中，需找到圆上每一点与原点的连线，这些连线构成圆锥。再用另一个平面（如y=1平面）截取该圆锥，即可得到相应的曲线。因此，椭圆和双曲线均可视为圆锥在不同平面截取下的结果。
  - 从特定角度观察时，无法看到与该平面平行的方向上的点。若用与y=1平面平行的平面截取圆锥，会发现某些点在截取过程中“消失”，因为这些点与原点的连线在截取平面之外。
- 圆锥曲线与无穷远点的关系 ﻿01:44:30﻿
  - 双曲线的两个点分别对应双曲线的两个无穷远点。双曲线加上两个无穷远点，相当于将双曲线扩展。双曲线的一支对应椭圆的一部分，从光源出发，双曲线上的每一个点与椭圆上的一个点对应，相应的线也是对应的。同样，椭圆的小半部分与双曲线的另一部分对应，但对应过程中漏掉了两个点，这两个点不在双曲线上，而是双曲线的两个无穷远点。双曲线实际上对应一个方向，所有的无穷远方向实际上经过某一点。无穷小方向一个对应一个方向，另一个对应另一个方向。这两个点分别对应，一支对应一个点，另一支对应另一个点。图表示它们是一一对应的关系。
  - 单点集对应的二次曲线是退化的，此处讨论的是非退化的情况，整个圆锥对应非退化的情况。复平面上的原和是复平面的一些焦点，复平面会出现虚的焦点，即虚数的焦点。在分类中，前面出现空集，后面出现空集的情况，在复平面看来，可能不是同一事物。复平面的理论比实的理论更加完备，分类更加清楚时，理论更好研究。
