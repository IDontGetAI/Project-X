# 研究生考试-计算机统考规划

## 目标

- 建立一套符合现有 taxonomy 的考研知识组织骨架。
- 将英语一、政治、数学一、408 计算机基础分别归入稳定学科路径。
- 把考试目标放在项目笔记中统一协调，而不是另建“考研”一级学科。

## 范围

- 包含英语一、政治、数学一、408 计算机基础四个科目。
- 包含资料归档入口、笔记归属建议、后续扩展路径。
- 暂时不包含具体真题录入、课程逐讲拆分与进度追踪细项。

## 科目归属

- 英语一 -> `language-learning/english`
- 政治 -> `social-science/political-science`
- 数学一 -> `mathematics/analysis`、`mathematics/algebra`、`mathematics/probability-statistics`
- 408 数据结构 -> `computer-science/algorithms-data-structures`
- 408 计算机组成原理 -> `computer-science/computer-architecture`
- 408 操作系统 -> `computer-science/systems`
- 408 计算机网络 -> `computer-science/networks-distributed`

- 已新增一级学科 `language-learning`，用于承接英语一及后续语言学习内容。
